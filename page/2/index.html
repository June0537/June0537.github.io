2024烟花三月下扬州

发表于 2024-04-01 更新于 2024-06-11 分类于 游记 本文字数： 21k

(20240329~0401)

扬州市，简称扬，古称江都、广陵，是江苏省下辖的地级市，位于江苏省中部，长江北岸。扬州地处江淮平原南部，长江三角洲北翼，西部比东部略高，有少量丘陵。长江流经南部边界，京杭运河穿越城区。市境西北部为高邮湖，是江苏省第三大湖(前二为太湖、洪泽湖)。扬州是中国首批国家历史文化名城，公元前486年吴王夫差于此筑邗城，为华夏九州之一，至今已2500余年。

2024千载伊阙地，神都洛阳城

发表于 2024-01-13 更新于 2024-06-09 分类于 游记 本文字数： 21k

(20240111~14)

洛阳是河南省下辖地级市，中原城市群的副中心城市，是河洛文化和华夏文明的重要发祥地。有十三个正统朝代，和二十多个政权将洛阳设为文化、经济、行政中心，拥有5000多年文明史、4000多年建城史和1500多年建都史，是中国历史上建都最早朝代最多、历时最长的城市。洛阳也是隋唐大运河的中心城市和丝绸之路的东方起点。
