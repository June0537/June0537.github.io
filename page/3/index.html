(20230213，泰山第43次行程)

12号早晨醒来，第一件事拿起手机看泰山直播，有点意外，雪还不小，中天门区域都银装素裹了，天外村/桃花峪大巴车停运了，但因为风不大，中天门索道还开着。

(20230117，泰山第41次行程)

2022悠然戏索道，拜会大将军

发表于 2022-08-21 更新于 2023-06-28 分类于 游记 本文字数： 6.5k

(20220820，泰山第38次行程)

泰山中天门索道之下，有条维修索道的野路，驴友称之为索道沟，可以近距离接触索道，中途还有几道梯子，惊险刺激，我打它的主意不是一天两天了，好几次都想走走索道沟，沟里那几道梯子让俺魂牵梦绕。

泰山东部条子梁上有座奇石山，天工开物，也叫小泰山，雄伟险峻，其上有将军石，也是著名的打卡地，许多人慕名去拜访大将军，一睹将军风采。
